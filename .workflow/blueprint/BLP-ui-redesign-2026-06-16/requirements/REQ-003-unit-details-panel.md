# REQ-003: 单位详情与指令面板

## 1. Description
选中单个或多个军事/平民单位后，在屏幕右下角固定区域显示的详细状态与操作面板。此面板特别要强化原本容易被玩家忽略的战斗关键数据。

## 2. User Story
As a 指挥官,
I want 能够在选中单位时一眼看到其当前的掩体防御加成和剩余护甲,
So that 我能做出正确的战术规避或集火决策，而不是单纯地拼血量。

## 3. Constraints & Rules
- 当选中多个异构单位时，面板 MUST 聚合并显示公共指令，而非崩溃或错误排版。
- 战斗隐式数值（如掩体加成/抗性）MUST 以显眼的高对比度图标和数值呈现，必要时辅以进度条或颜色警告（如破甲红色警告）。
- 所选单位的生命值/能量值 MUST 以平滑插值过渡（Lerp）而非瞬间跳跃的方式表现数值减少。

## 4. Acceptance Criteria
1. **单选展示**: 选中一个载具，能明确看到其当前的装甲值和受地形影响的掩体系数。
2. **多选展示**: 框选 10 个步兵和 2 辆坦克，右下角能切换到编组概览模式，显示各兵种总数量。
3. **指令下达**: 面板附带的攻击/移动指令按钮能够正确响应，并反馈到单位行为树。
